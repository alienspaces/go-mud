## Dungeons

**List dungeons:**

- [Response Schema](server/src/controllers/dungeon/schema/dungeon.schema.json)

```bash
GET /api/v1/dungeons
```

**Get a dungeon:**

- [Response Schema](server/src/controllers/dungeon/schema/dungeon.schema.json)

```bash
GET /api/v1/dungeons/{:dungeon_id}
```
